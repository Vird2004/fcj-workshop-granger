#### What you need before starting

To successfully complete this workshop, please ensure you have the following ready:

1. **AWS Account**: An active AWS account with Administrator access. If you don't have one, you can create a Free Tier account.
2. **AWS CLI (Optional but recommended)**: Installed and configured on your local machine to easily upload files to S3 or deploy Lambda functions.
3. **Source Code**: You should have the LunaGenZ source code downloaded locally. This includes:
   - The Frontend web files (HTML/CSS/JS).
   - The Backend logic code (Node.js or Python) for the Numerology calculations.
   - The PDF generation script.
4. **Code Editor**: A text editor like VS Code installed to view and edit the source code if necessary.

Once you have prepared all the prerequisites, proceed to the next step to deploy the Backend!
