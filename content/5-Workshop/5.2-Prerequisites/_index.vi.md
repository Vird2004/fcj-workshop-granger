#### Các yêu cầu trước khi bắt đầu

Để hoàn thành tốt workshop này, vui lòng đảm bảo bạn đã chuẩn bị sẵn:

1. **Tài khoản AWS**: Một tài khoản AWS đang hoạt động với quyền truy cập Administrator. Nếu chưa có, bạn có thể tạo một tài khoản Free Tier.
2. **AWS CLI (Tùy chọn nhưng khuyến khích)**: Đã cài đặt và cấu hình trên máy tính cá nhân để dễ dàng tải file lên S3 hoặc deploy Lambda.
3. **Mã nguồn (Source Code)**: Bạn cần tải mã nguồn dự án LunaGenZ về máy. Mã nguồn bao gồm:
   - Các file giao diện Frontend (HTML/CSS/JS).
   - Code Backend (Node.js hoặc Python) xử lý tính toán Thần Số Học.
   - Script tự động tạo PDF.
4. **Trình soạn thảo mã (Code Editor)**: Ví dụ như VS Code, để mở và chỉnh sửa mã nguồn khi cần thiết.

Sau khi đã chuẩn bị đầy đủ các yêu cầu trên, hãy chuyển sang bước tiếp theo để bắt đầu triển khai Backend nhé!
