# Deploy LunaGenZ (Numerology) on AWS Serverless

#### Overview

**LunaGenZ** is a Numerology application built entirely on an AWS Serverless architecture. This workshop will guide you through the process of deploying the backend logic, automating PDF report generation, and hosting the frontend.

In this lab, you will learn how to:
+ Structure and deploy an AWS Serverless Backend using AWS Lambda and API Gateway.
+ Automate the generation of internal PDF reports for Numerology calculations.
+ Host a high-availability Frontend application on Amazon S3 and Amazon CloudFront.
